## 📸 Overview

**HexaVision Enterprise v9.0** is a professional-grade **Image Forensics and Analysis Suite** designed for cybersecurity professionals, forensic investigators, and security researchers. It provides a comprehensive set of tools to analyze digital images, detect manipulations, extract hidden metadata, and identify the exact camera device used to capture a photo.

### 🎯 Key Capabilities
- ✅ **EXIF Metadata Extraction** – Camera settings, timestamps, device info
- ✅ **GPS Geolocation Analysis** – Extract and map exact coordinates
- ✅ **Error Level Analysis (ELA)** – Detect image manipulation and editing
- ✅ **Thumbnail Extraction** – Recover hidden thumbnails from JPEG files
- ✅ **Reverse Image Search** – Search on Google, TinEye, Yandex, Bing
- ✅ **Face Detection** – Identify faces in images using OpenCV
- ✅ **Face Matching** – Compare two faces with similarity scoring
- ✅ **Steganography Detection** – Find hidden data embedded in images
- ✅ **PRNU Camera Fingerprinting** – Identify exact camera device (NEW!)
- ✅ **Timeline Analysis** – Track file creation and modification history
- ✅ **Deep GPS Analysis** – Get city, state, country from coordinates
- ✅ **HTML/JSON Export** – Generate professional forensic reports
