## 📸 **About HexaVision**

**HexaVision Enterprise v9.0** is a professional **Image Forensics and Analysis Suite** for cybersecurity experts, forensic investigators, and security researchers. It provides 14 powerful tools to analyze digital images, detect manipulations, extract hidden metadata, and identify the **exact camera device** used to capture a photo.

## 🔬 **PRNU Camera Fingerprinting (NEW!)**

**What is PRNU?**  
Every camera sensor has a unique **noise pattern** called Photo Response Non-Uniformity (PRNU). It's like a **digital fingerprint** for cameras!

**How it works:**

**What you can do:**
- ✅ Extract unique noise fingerprint from any image
- ✅ Save reference fingerprints from known cameras
- ✅ Compare images to check if they came from the **SAME camera device**
- ✅ Detect fake/edited images by mismatched fingerprints

**Use Cases:**
- 🔍 Identify the exact camera in forensic investigations
- 🕵️ Link multiple images to the same device
- 🎨 Detect AI-generated or heavily edited images
- 📱 Verify social media profile authenticity
